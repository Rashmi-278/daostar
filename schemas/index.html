DAO Schema

name name of the DAO
description free text, details regarding the DAO
membersURI a URI pointing to a Json file representing the Members object
proposalsURI a URI pointing to a Json file representing the Proposals object
activityLogURI a URI pointing to a Json file representing the Activity Log object
governanceURI a URI pointing to a flat file, detailing the governance system of the DAO
issuerURI a URI pointing to a Json file, detailing the attestation issuers for the DAO

Members Schema

name name of the DAO
members array of DAO member objects. A member object can include any relevant properties which relate to the specific DAOs concept of membership. An example of a simple member object will include one property which is an Ethereum address.

Proposals Schema

name name of the DAO
proposals array of DAO proposal objects. A proposal object must include an ID property as described in the Proposal Schema. Additionally can include any relevant properties which relate to the specific DAOs concept of a proposal.

Activity Log Schema

name name of the DAO
activities array of DAO activity objects. An activity object represents the interplay between members and proposals.

Proposal Schema

id every unique on-chain proposal MUST be associated to a proposal ID of the form CAIP10_ADDRESS + “?proposalId=” + PROPOSAL_COUNTER, where PROPOSAL_COUNTER is an arbitrary string which is unique per CAIP10_ADDRESS
name name of the proposal
contentURI a URI pointing to some content such as a forum post or a description on a voting platform which predates or accompanies the actual proposal
status free text describing the current status of the proposal
calls for on-chain proposals with non-empty execution, an array of Call objects, which detail the execution data. The main use-case for this data is execution simulation of proposals. An example of a Call object is the CallDataEVM object described below.

Activity Schema

id activity id
proposal an object containing the id of the proposal which the activity is related to
member The member object which for which the activity is related to

CallDataEVM Schema

operation call or delegate call
from from address
to to address
value call value
data call data
